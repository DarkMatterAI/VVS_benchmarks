# 1. Download Datasets and Create Artifacts

**Prerequisites:** [System Requirements](00_system_requirements.md)

All datasets used in the paper are downloaded from external sources. Code in
`create_artifacts` downloads the relevant datasets, performs light preprocessing,
and creates the datasets used for training the Embedding Compression and Embedding
Decomposer models.

## Downloaded datasets

- ZINC15 10M dataset ([ref](https://deepchem.readthedocs.io/en/latest/api_reference/moleculenet.html#zinc15-datasets))
- Lyu 140M dataset ([ref](https://pubmed.ncbi.nlm.nih.gov/30728502/)), referred to as "D4" in the codebase
- Enamine building blocks and SMARTS reactions from the SyntheMol paper ([ref](https://www.nature.com/articles/s42256-024-00809-7))
- SyntheMol random forest models ([ref](https://www.nature.com/articles/s42256-024-00809-7))
- COCONUT natural products dataset ([ref](https://pubmed.ncbi.nlm.nih.gov/33423696/))
- Thompson Sampling GitHub repo ([ref](https://pubs.acs.org/doi/10.1021/acs.jcim.3c01790))
- ChEMBL EGFR IC50 data ([ref](https://www.ebi.ac.uk/chembl/))

## Created artifacts

After downloading, the following datasets are created within the repo:

- **Enamine Assembled dataset** — `{(bb1, bb2), (product)}` triples generated by
  computational synthesis using the downloaded building blocks and reactions.
- **Embedding Compression training dataset** — SMILES sampled from the ZINC 10M,
  Lyu 140M, and Enamine Assembled datasets.

## Instructions

To download and create all datasets, run:

```bash
cd src/create_artifacts
./run_download_files.sh
```

> **Note:** This step is slow and downloads/produces a large volume of data.

> **Tip:** Exact CSV files of the Enamine Assembled and Embedding Compression
> datasets used in this work are available in the
> [Zenodo data archive](https://zenodo.org/uploads/18615633), and can be used in
> place of regenerating them.
